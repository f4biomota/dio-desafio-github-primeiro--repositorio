Comandos GIT
============

git clone - baixa o projeto inteiro do repositório remoto.
git init - Inicia o versionamento de uma pasta nova.
git status - Mostra o status do diretório e seus arquivos com as pastas filhas.
git add . - Adiciona todos os arquivos e pastas não versionados para serem monitorados.
git push - Envia os arquivos já comitados para o repositório remoto.
git pull - baixa os arquivos do repositório remoto.
git commit - Comita os arquivos e os deixa prontos para serem enviados para repositório remoto.
